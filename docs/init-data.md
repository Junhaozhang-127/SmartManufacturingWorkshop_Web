# 初始化数据与权限账号

## 初始化步骤

```bash
corepack pnpm db:generate
corepack pnpm db:migrate:dev
corepack pnpm db:seed
```

## 种子数据内容

- 组织架构与基础角色
- RBAC 权限与菜单
- 审批模板
- 示例成员、设备、耗材、经费账户、成果与考核方案
- 演示通知与审批数据

## 默认账号

- `teacher01 / 123456`
  角色：教师
  范围：全局
- `leader01 / 123456`
  角色：实验室负责人
  说明：首登改密流程验证账号
- `minister01 / 123456`
  角色：部长
  范围：部门级
- `hybrid01 / 123456`
  角色：部长、组长
  用途：角色切换与数据范围联调
- `member01 / 123456`
  角色：成员
  范围：本人参与

## 权限验证建议

- 教师账号：全链路查看与审批回归
- `hybrid01`：验证同一用户切换角色后的菜单、列表和待办差异
- `member01`：验证最小权限与本人数据范围

## 注意

- 以上账号仅用于开发/测试环境
- 若重置数据库，需重新执行 `pnpm db:seed`
